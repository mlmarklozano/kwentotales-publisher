Kwentotales Publisher
TikTok Content Publishing Tool
About

Automated short-form video publishing for Kwentotales

Kwentotales Publisher is a private automation tool that publishes short-form video content from the Kwentotales channel to TikTok. It is operated exclusively by the Kwentotales team and is not available to third-party users.

The tool manages the full pipeline — from video preparation to upload — for the @kwentotales TikTok account, which produces Filipino and Southeast Asian mystery and true crime content.

TikTok API Usage

How this app uses TikTok permissions

This tool requests only the permissions necessary to upload video content on behalf of the account owner.

video.upload
Upload video files to the creator's TikTok inbox for review and publishing. Videos are uploaded only to the @kwentotales account.
user.info.basic
Read basic profile information to verify account identity during authorization. No personal data is stored.
Data & Privacy

Data handling

Kwentotales Publisher does not collect, store, or share personal data from TikTok users. OAuth tokens are stored securely on a private server and used solely to authenticate API requests for the @kwentotales account.

Video content published through this tool is original content created and owned by Kwentotales. No user data from TikTok is retained beyond the active session.
