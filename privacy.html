Kwento Tales Publisher

Privacy Policy

Effective date: July 15, 2025  ·  Last updated: July 15, 2025

Summary: This App does not collect, store, or share any personal data from TikTok users. It is a private internal tool used solely to publish videos to the operator's own TikTok account.

1. What This App Is

Kwento Tales Publisher is a private, internal automation tool. It connects to TikTok's Content Posting API to upload pre-produced video content to a single TikTok account owned by the operator. It is not a public application and has no end users other than its operator.

2. Data We Access

Through TikTok's API, the App accesses only the following information belonging to the operator's own TikTok account:

  • Account credentials (OAuth access and refresh tokens) used to authenticate API requests
  • Upload endpoints required to post video files
  • Basic account metadata needed to confirm successful publication

3. Data We Do Not Collect

The App does not collect, process, or store any data from:

  • TikTok users who view, like, comment on, or follow the operator's content
  • Any third-party individuals or accounts
  • Audience analytics, follower information, or engagement metrics

4. How Data Is Used

OAuth tokens are stored locally on the operator's private server solely for the purpose of authenticating API calls. No data is transmitted to any third party other than TikTok's official API endpoints as required to perform uploads.

5. Data Retention

Access tokens are retained locally only as long as necessary for automated publishing operations. Tokens can be revoked at any time through TikTok's account security settings.

6. Third-Party Services

The App communicates exclusively with TikTok's official API (api.tiktok.com). No other third-party services receive data through this App.

7. Security

Credentials are stored in environment variables on a private server not accessible to the public. No credentials are stored in source code or transmitted over unencrypted connections.

8. Your Rights

Since no personal data from third parties is collected, there is no personal data to access, correct, or delete on your behalf. TikTok account access can be revoked at any time through TikTok's connected apps settings.

9. Changes to This Policy

This policy may be updated at any time. The effective date at the top of this page reflects the most recent revision.
